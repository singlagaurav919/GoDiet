About

Objective

The goal of our web application is to track nutritional content of what you consume with a simple snap of your food item. The website processes the image of a food item, retrieves nutritional content of the food item.

Tech Stack

We used Python (Flask) for our API. We have used CloudSight and NutritionX API.Other technical knowledge which was required was that of HTML5, CSS, Python, Flask, JavaScript

User Flow

We built our core website on Flask(Python) and hosted it on Heroku. The image is sent as multipart data from website as a POST request to /upload endpoint which calls CloudSIght API to retrieve food item name and then NutritionX API is called to retrieve relevant nutritional data of that item.
